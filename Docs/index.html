Summary: Latitude vs. X

How does weather change as you get closer to the equator? The goal of the Latitude analysis was to visualize how weather changes as a location gets closer in distance to the equator. Weather data was pulled using a series of API calls from the OpenWeatherMap website for more than 500 cities around the world.

After merging and cleaning up the dataset, Matplotlib was used to create scatter plots of latitude versus different aspects of weather. The weather factors used for each city were maximum temperature, humidity, cloudiness, and wind speed. This website provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of observed trends and correlations.
